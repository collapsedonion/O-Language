#  Variables

## Basic info

Variable can be every of existing type, data for variable allocated at the stack

## Variable definition

To define variable must be used "var" compiler instruction

### VAR prototype
 ```
 var:[data-type] <name>
 ```
 
### Example
    
    var:bool boolTypeVar;
       
    var:int intTypeVar;
        
    var:float floatTypeVar;
